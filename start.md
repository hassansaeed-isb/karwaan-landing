# Quick Start Guide

## Installation & Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm start
   ```

3. **Open your browser:**
   Navigate to `http://localhost:3000`

## What You'll See

Your fully responsive Karvaan landing page with:

- **Mobile-optimized navigation** with hamburger menu
- **Responsive hero section** with gradient text effects
- **Problem/solution sections** that stack properly on mobile
- **Feature cards** that adapt to screen size
- **Call-to-action sections** optimized for touch
- **Smooth animations** and hover effects
- **Scroll-to-top button** for easy navigation

## Mobile Testing

Test on different screen sizes:
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px  
- **Desktop**: > 1024px

Use browser dev tools to simulate mobile devices or test on actual devices.
